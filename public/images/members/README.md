# Member Images

Add member photos to this folder using the following naming convention:

**Format:** `member_name.jpg`

- Use **lowercase** letters
- Replace spaces with **underscores**
- Use `.jpg` extension

## Examples
- `ayush_kamble.jpg`
- `john_doe.jpg`
- `priya_sharma.jpg`

The website will automatically match these images to members from the database by their name.
